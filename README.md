# Naradji

**The store you finish in one breath.**

Built at [Cursor Hackathon Ahmedabad](https://cursor.com).

Naradji is a voice-first Indian daily-needs storefront. Speak a grocery list in Gujarati, Hinglish, or any Indian language — the store morphs to the right products, prefills quantities and payment, and places a COD order. Zero taps, one confirm.

## The idea

While chat agents pull shopping out of the store, Naradji makes the **store itself** the agent. You talk to the store; the storefront regenerates around you.

## Stack (planned)

- Next.js (App Router)
- Payload CMS (catalog + orders)
- Vercel AI SDK
- Sarvam STT (Indian languages / code-mix)
- ElevenLabs or Sarvam TTS
